# SecuRock Community Edition vs. Enterprise

To drive adoption and product-led growth, SecuRock offers a robust Community Edition for individual researchers and small teams.

## Feature Comparison

| Feature | Community Edition (Free) | Enterprise (Paid) |
| :--- | :--- | :--- |
| **Deployment** | Docker / Self-Hosted | Cloud SaaS / On-Prem HA |
| **Data Retention** | 3 Days | 1 Year+ |
| **Users** | 1 User | Unlimited / SSO |
| **AI Detection** | Standard Models | Custom Re-training |
| **Threat Intel** | Open Source Feeds | Premium Feeds (CrowdStrike, etc.) |
| **SLA** | Community Support | 24/7 Dedicated Support |
| **Compliance** | None | SOC2, HIPAA, GDPR Reports |
| **Attack Map** | Standard View | Real-time Global War Room |

## Community Edition Strategy
1.  **Open Core**: The core ingestion and detection engine is open source.
2.  **Upgrades**: Easy license key upgrade unlocks Enterprise features (SSO, Reporting) without re-installation.
3.  **Telemetry**: Optional anonymous usage stats helps improve AI models for everyone.
